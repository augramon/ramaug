# Meu Portfólio

Este é o meu portfólio. Criei este projeto para apresentar meus trabalhos, minhas habilidades e a minha evolução como desenvolvedor. O foco aqui é entregar uma experiência rápida, moderna e visualmente agradável

---

## Sobre o Projeto

Desenvolvi este portfólio para mostrar de forma simples e objetiva:
- quem eu sou  
- o que eu faço  
- meus principais projetos  
- minha stack e meu estilo de desenvolvimento  

O site é leve, responsivo e pensado para causar uma boa impressão

Pretendo mantê-lo constantemente atualizado
